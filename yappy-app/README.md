# yappy-app

Tauri 2 + SvelteKit shell for [Yappy](../README.md).

See the [project README](../README.md) and [CONTRIBUTING](../CONTRIBUTING.md) at the repo root.

```bash
npm install
npm run tauri dev    # dev with hot-reload
npm run tauri build  # production build (or ../scripts/build-mac.sh on macOS)
```
