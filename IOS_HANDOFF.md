## 🔴 What's still on you

The ASC API key can register apps but Apple blocks **CREATE** on `/apps` for new bundle IDs. Two 5-minute steps:

### Step 1 — Apple Developer portal (https://developer.apple.com/account/resources/identifiers/list)

1. **Identifiers → App IDs → `com.yappy.app`**: enable
   - ☑ App Groups
   - ☑ Increased Memory Limit

2. **Identifiers → App IDs → `com.yappy.app.ShareExtension`** (auto-created by Xcode on first build of the share-extension target): enable
   - ☑ App Groups

3. **Identifiers → App IDs → `com.yappy.app.YappyWidgets`** (same — auto-created): enable
   - ☑ App Groups

4. **Identifiers → App Groups → Register**:
   - Identifier: `group.com.yappy.app`
   - Description: Yappy shared container
   - Then add this group to **all three App IDs** above.

### Step 2 — App Store Connect (https://appstoreconnect.apple.com/apps)

1. Click **+** → **New App** with:
   - Platforms: iOS
   - Name: **Yappy**
   - Primary Language: English (U.S.)
   - Bundle ID: `com.yappy.app`
   - SKU: `yappy-ios`
   - User Access: Full Access

When both steps are done, run:

```bash
node scripts/asc-helper.mjs status
```

Should print `✓ App found: Yappy (id=...)`. Then:

```bash
./scripts/ios-build-and-upload.sh
```

builds + signs + uploads the IPA, polls ASC for processing completion, creates a public Beta group, attaches the build, and submits for Beta App Review. Public TestFlight link comes out the other side after 24–48h Apple review.

## Architecture cheat sheet

```
yappy-app/
├── src-tauri/
│   ├── src/
│   │   ├── mobile.rs            # FFI to Swift (audio/activity/share queue)
│   │   ├── commands.rs          # render_audiobook_cmd → BackgroundAudioGuard + activity_start
│   │   └── ...
│   └── gen/apple/               # Xcode project (generated by xcodegen from project.yml)
│       ├── project.yml          # Source of truth — bundle IDs, entitlements, extensions
│       ├── Sources/yappy-app/
│       │   ├── AudioSession.swift          # Silent BG audio + Share queue drain
│       │   ├── LiveActivityBridge.swift    # ActivityKit start/update/end
│       │   └── RenderActivityAttributes.swift  # Shared with widget target
│       ├── YappyWidgets/                   # Live Activity widget extension
│       └── YappyShareExtension/            # Share Sheet extension
└── src/lib/platform.ts          # Reactive $isIOS / $platformName stores

vendor/ort-ios/                  # ONNX Runtime 1.22.0 XCFramework (gitignored)
scripts/
├── fetch-ort-ios.sh             # Re-fetch ORT + regen .cargo/config.toml
├── asc-helper.mjs               # ASC REST API helper (status / pipeline / submit)
└── ios-build-and-upload.sh      # One-shot: build → sign → upload → TestFlight
maestro/flows/                   # End-to-end iOS sim tests
```
